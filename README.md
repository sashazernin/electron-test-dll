# electon-test-dll

Electron приложение с React для тестирования DLL.

## Установка

Перед началом работы необходимо установить зависимости:

```bash
npm install
```

## Скрипты

### `npm run dev`

Запускает приложение в режиме разработки:

- Запускает React dev server на `http://localhost:3000`
- Одновременно запускает Electron окно, которое подключается к dev server
- Изменения в коде автоматически перезагружаются

### `npm run dist`

Собирает приложение для production и создаёт установщик:

1. Собирает React приложение в папку `build` (оптимизированная production сборка)
2. Упаковывает приложение в exe файл через `electron-builder`
3. Результат сохраняется в папку `release`:
   - `release/electon-test-dll Setup 0.1.0.exe` - установщик
   - `release/win-unpacked/electon-test-dll.exe` - распакованная версия для тестирования

## Функции DLL

Приложение поддерживает вызов функций из DLL:

### `addition`

Функция для сложения чисел:

- Первый параметр: `int` (целое число)
- Второй параметр: `float` (число с плавающей точкой)
- Возвращает: `float`

### `multiplication`

Функция для умножения чисел:

- Первый параметр: `int` (целое число)
- Второй параметр: `float` (число с плавающей точкой)
- Возвращает: `float`
